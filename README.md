# TuSecurity Club Platform 🛡️

منصة نادي الأمن السيبراني (TuSecurity - جامعة Taif University).
منصة تفاعلية تهدف إلى تعزيز الوعي وبناء مجتمع سيبراني محترف من خلال التعليم، التحديات، والمحاكاة.

## ✨ المميزات الرئيسية

- **نظام تعليمي متكامل (LMS)**: مسارات تعليمية، دروس تفاعلية، وتتبع التقدم.
- **محاكيات سيبرانية (Simulators)**:
  - محاكي الطرفية (Terminal Simulator) لتعلم أوامر Linux.
  - محاكي الهجمات والدفاع (Cyber Range).
  - تحديات CTF.
- **لوحة تحكم (Dashboard)**: ملف شخصي للطالب، إحصائيات، وجوائز.
- **نظام الرتب والجوائز**: تحفيز الطلاب من خلال نقاط XP والأوسمة.
- **مجتمع تفاعلي**: أخبار النادي، الفعاليات، واللجان.

## 🛠️ التقنيات المستخدمة

### Frontend (الواجهة الأمامية)
- **React.js**: مكتبة بناء الواجهات.
- **Vite**: أداة بناء وتطوير سريعة.
- **Tailwind CSS**: إطار عمل للتصميم.
- **Framer Motion**: مكتبة للرسوم المتحركة.
- **Lucide React**: أيقونات عصرية.

### Backend (الخلفية)
- **Node.js**: بيئة تشغيل JavaScript.
- **Express.js**: إطار عمل للخادم.
- **SQLite**: قاعدة بيانات خفيفة وسريعة.

## 🚀 تشغيل المشروع محلياً

### المتطلبات
- Node.js (v16 أو أحدث)
- npm

### 1. تثبيت الاعتماديات
```bash
# تثبيت مكاتب الواجهة الأمامية
npm install

# تثبيت مكاتب الواجهة الخلفية
cd backend
npm install
cd ..
```

### 2. إعداد متغيرات البيئة
قم بإنشاء ملف `.env` في مجلد `backend` وأضف المتغيرات اللازمة (مثل مفاتيح التشفير).

### 3. تشغيل المشروع
يمكنك تشغيل المشروع بالكامل باستخدام السكربت المخصص:
```bash
./start-app.sh
```

أو تشغيل كل جزء على حدة:

**تشغيل الخادم (Backend):**
```bash
cd backend
node server.js
```

**تشغيل الواجهة (Frontend):**
```bash
npm run dev
```

## 📂 هيكلية المشروع

- `/src`: ملفات الواجهة الأمامية (React).
- `/backend`: ملفات الخادم وقاعدة البيانات.
- `/public`: الملفات العامة والصور.
